# Gravity Escape

A futuristic neon anti-gravity arcade game inspired by Google Antigravity, Geometry Dash, and cyberpunk visuals.

## Live Demo

[https://krishkumar7982.github.io/gravity-escape/](https://krishkumar7982.github.io/gravity-escape/)

---

## Features

* 4 Difficulty Modes:

  * Easy
  * Medium
  * Hard
  * Very Hard
* Reverse gravity using the Space key
* Neon cyberpunk UI and glowing effects
* Infinite obstacle generation
* Increasing speed and difficulty over time
* Lives system and invincibility timer
* AI Auto Play mode
* High score saving using localStorage
* Unlock system for harder difficulties
* Animated menus and level complete screens

---

## Controls

| Key   | Action          |
| ----- | --------------- |
| Space | Reverse Gravity |
| P     | Pause Game      |
| R     | Restart         |

---

## Difficulty Modes

### Easy

* Slow speed
* 3 lives
* Large obstacle gaps

### Medium

* Faster speed
* Moving obstacles
* 2 lives

### Hard

* Fast gameplay
* Random obstacle patterns
* 1 life

### Very Hard

* Extreme speed
* Fake gaps
* Rapid obstacle spawning
